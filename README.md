# This is a static mockup of a Marketing Agency Landing Page
built based on Figma Template "Marketing Agency Landing Page" by Aqsath Faza

# Notes
* The project has modular structure with imported font "Alexandria" from Google Fonts
* The elements are also modular: basic style of every element is located in null.css file and modified with other style files or attribute "mod" (shortened from "modifier")
* BEM is not used everywhere. In some cases, where BEM could be used, .class element was utilized for convinience on coding
* Expecting the width to be atleast 360px, the site is fully adaptible
